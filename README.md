# Python Text-Adventure (Peithorn)

## Grundidee: 
Ein Textbasiertes-Fantasy-Adventure Rollenspiel mit der Möglichkeit verschiedene Wege zu gehen, 
beziehungsweise verschiedene Auswahlmöglichkeiten zu haben.
Gesteuert wird das ganze durch Fragen die dem Spieler gestellt werden und dieser muss dann den von ihm bestimmten Weg bzw. 
die von ihm bestimmte Entscheidung durch Eingabe der bestimmten Zahl entscheiden. 
Anschließend springt das Spiel zum ausgewählten Schauplatz oder die ausgewählte Entscheidung wird ausgeführt. 


## Die Geschichte spielt im Königreich PEITHORN.
Man wird von einem zwielichtigen Mann auf eine Reise geschickt um ein Monster in einer Höhle auf der anderen Seite von 
Peithorn zu beseitigen. 
Auf dieser Reise wird der Spieler des Öfteren auf die Wahl gestellt welchen Weg er geht oder welche Entscheidung er trifft. 
Dabei begegnen ihm einige Schauplätze wie ein düsterer Wald, ein dunkler Pfad, eine Lichtung, die tiefen des Waldes, 
ein Hexenhaus, ein gefährliches Gebirge, eine Schlucht, ein Tal, ein Sumpf und die Höhle.
Außerdem trifft der Spieler auf verschiedene Charaktere wie den zwielichtigen Mann, einen Troll, einen Zwerg, eine Hexe, 
ein Monster und einem Drachen.


## Steuerung: 
Tasten 1, 2, 3 und dann Enter. Alle anderen Eingaben beenden das Spiel.
